# 14-day no-card trial with Shopify-prefill onboarding

Signup requires no credit card — the friction of a card form at signup measurably hurts trial conversion, and the 14-day window is short enough that abuse risk is bounded. The onboarding flow funnels through "Connect Shopify" early; on connect we prefill **Store** name, currency, timezone, **AccountOwner** contact, hours of operation, and seed one Taime **Location** per Shopify Location. Shopify staff are NOT auto-imported as **Employees** — the boutique manager invites them deliberately so we don't create unwanted **StoreMemberships** or surprise people with email invites. After day 14 without a chosen **Plan**, admin features lock but clock-in/out remains available so we never block timekeeping (a labor-compliance concern); existing **TimeEntries** stay readable so payroll can still close.
